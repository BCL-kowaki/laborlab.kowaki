# お知らせ（News）の書き方ガイド

このフォルダ（`content/news/`）に **`.mdx` ファイル**を新規作成すると、サイトの「お知らせ」ページに自動で記事が掲載されます。

---

## ステップ1: ファイルを作る

このフォルダに新しいファイルを作ります。

```
content/news/[ファイル名].mdx
```

**ファイル名のルール:**

- 半角英数字とハイフン（`-`）のみ使用
- 拡張子は `.mdx`
- 例: `childcare-leave-law-revision.mdx`、`seminar-2024-12.mdx`

> **重要**: ファイル名がそのまま記事のURL（`/news/childcare-leave-law-revision` など）になります。日本語は使わないでください。

---

## ステップ2: 記事の中身を書く

`_template.mdx.example` を参考にして書きます。

最低限必要な内容:

```mdx
---
title: 記事のタイトル
date: 2024-09-15
category: 制度改正
excerpt: 一覧で表示される短い説明文。
---

ここから本文を書きます。
```

### `---` で囲まれた部分（フロントマター）の説明

| 項目 | 必須 | 説明 |
|---|---|---|
| `title` | ◎ | 記事のタイトル |
| `date` | ◎ | 公開日（YYYY-MM-DD 形式） |
| `category` | ◎ | カテゴリ（後述の4種類のうち1つ） |
| `excerpt` | ◎ | 一覧ページに表示される短い説明文（120文字程度推奨） |

### カテゴリの種類

以下の4つから選んでください。**正確にこの文字を入力**してください（カッコや別表記は不可）。

| カテゴリ | 用途 | 表示色 |
|---|---|---|
| `お知らせ` | 通常のお知らせ | 金色のバッジ |
| `制度改正` | 法改正・制度変更の告知 | **赤色のバッジ（強調）** |
| `セミナー情報` | セミナー・イベント告知 | 青系のバッジ |
| `コラム` | 短い記事・読み物 | ネイビーのバッジ |

---

## ステップ3: 本文を書く

フロントマター（`---` で囲まれた部分）の下に本文を書きます。

### 段落

```
1つ目の段落です。

1行空けると2つ目の段落になります。
```

### 見出し

```
## 大見出し（H2）

### 中見出し（H3）
```

### 太字・斜体

```
**太字にしたい部分**
*斜体にしたい部分*
```

### 箇条書き

```
- 項目1
- 項目2
- 項目3
```

### 番号つきリスト

```
1. 1つ目
2. 2つ目
3. 3つ目
```

### リンク

```
[リンクの文字](https://example.com)
[料金ページへ](/pricing)
```

### 引用（注意書きなど）

```
> ここは引用ブロックです。
> 注意書きを目立たせたい時に使います。
```

---

## ステップ4: 公開する

ファイルを保存すると、Takuya が公開作業（git push）を行います。
公開後、自動的にサイトの「お知らせ」ページに反映されます。

---

`childcare-leave-law-revision.mdx` を参考にしてください。実際に公開されている記事の元データです。

---

## 困ったときは

ファイルの書き方が分からない、表示がおかしい、などあれば Takuya にお問い合わせください。

**Tips:**
- 内容に自信がない場合は `category: 制度改正` ではなく `category: お知らせ` で投稿し、Takuya のレビューを受けてから差し替えるのも安全です。
- 法改正情報を投稿する際は、必ず**厚生労働省などの一次情報**を確認してください。
