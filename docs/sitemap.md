## SEO構造

### パンくずリスト

全ページにBreadcrumbListを実装：

```
ホーム > サービス > 顧問契約
```

### JSON-LD構造化データ

- `Organization`（事務所情報）: 全ページ
- `LocalBusiness`: About、Contact
- `Service`: Services詳細
- `Article`: Blog記事
- `BreadcrumbList`: 全ページ
- `FAQPage`: FAQページ

### sitemap.xml

Next.js の `app/sitemap.ts` で動的生成：

```typescript
// src/app/sitemap.ts
export default function sitemap() {
  return [
    { url: 'https://laborlab.kowackey.com', lastModified: new Date(), priority: 1.0 },
    // 以下、全ページを列挙
  ];
}
```

### robots.txt

```
User-agent: *
Allow: /

Sitemap: https://laborlab.kowackey.com/sitemap.xml
```
